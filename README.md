상점 페이즈와 실행 페이즈로 구분됨

1. 상점 페이즈
    - 진입할 때마다 일정량의 골드 지급
    - 골드로 시리얼의 개수를 늘리거나 등급을 강화할 수 있는 카드 구매 가능
    - 실행 페이즈에서 사용 가능한 액티브 슬롯을 늘리거나 카드 리롤도 가능
2. 실행 페이즈
    - 구매한 카드를 원하는 만큼 실행 슬롯에 옮기고 버튼을 누르면, 현재 가진 시리얼에 해당 카드의 효과가 적용됨
    - 일정 점수 이상이면 통과, 아니라면 패배
